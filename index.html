    WebGIS Weather Application

    This weather app was developed using open-source web mapping libraries like Leaflet.js, along with HTML, CSS, and Vanilla JavaScript. To display weather information, an API integration was implemented to retrieve the current temperature, wind speed, humidity, and the forecast for the upcoming days in a city of interest. When the name of a city is entered, the app zooms in on the geographical location of the city. This app is designed to be useful for travel planning, environmental monitoring, and survey planning, among other applications.
